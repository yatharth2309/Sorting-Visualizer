# Sorting-Visualizer

Visualization of different sorting algorithms in C++ with SDL2 Library.

Using this Sorting Visualizer, we will visually analyse the working of 6 Sorting Algorithms.
#### 1. Selection Sort
#### 2. Insertion Sort
#### 3. Bubble Sort
#### 4. Merge Sort
#### 5. Quick Sort
#### 6. Heap Sort
The list size is fixed to 130 elements.

A randomized list can be generated any time and any of the above mentioned sorting algorithms can be used to sort that random list.

Here, all sorting algorithms will sort the elements in ascending order. 

Since the max list size (130) is very small, faster algorithms like Merge Sort, Quick Sort etc have been delayed for proper visualization.

# How to run

You can run Sorting Visualizer using the C++ source code available in the repository i.e., sorting_visualizer.cpp but you will have to install and setup the SDL2 library first. 

Follow Lazy Foo' Productions' tutorial to setup SDL2 Library. -> [How to download and setup SDL2 library](http://lazyfoo.net/tutorials/SDL/01_hello_SDL/index.php).

# Controls

WARNING: Giving repetitive commands may cause latency and the visualizer may behave unexpectedly. Please give a new command only after the current command's execution is done.

Available Controls inside Sorting Visualizer:-
- Use 0 to Generate a different randomized list.
- Use 1 to start Selection Sort Algorithm.
- Use 2 to start Insertion Sort Algorithm.
- Use 3 to start Bubble Sort Algorithm.
- Use 4 to start Merge Sort Algorithm.
- Use 5 to start Quick Sort Algorithm.
- Use 6 to start Heap Sort Algorithm.
- Use q to exit out of Sorting Visualizer
